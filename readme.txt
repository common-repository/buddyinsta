BuddyInsta plugin used to show instagram images in your buddypress profile.

== Description ==

BuddyInsta plugin used to show instagram images in your buddypress profile using member's instagram username.


A few notes about the sections above:

*  It show instagram images.
*  It show images from your instagram username.
*  Images shows at profile tab in Instagram Photos subtab.

   

== Installation ==
1. Downalod plugin from here and install in plugin directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Setting->BuddyInsta in wp-admin section and put your Client ID and Client Secret and submit it.
4. Now,connect your profile with Instgram.Click on "Sign in with Instagram" button at frontend in Setting->Instagram.
5. Automatically it will takes your instagram photos.



== Frequently Asked Questions ==

= Where will images showing =

Images will show in member's buddypress profile.It will create a sub menu in members profile section.

= How to get instagram client id and client secret =

Go at http://instagram.com/developer/ 
click on "Register Your Application" button and follow step by step process.
